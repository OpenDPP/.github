# Security Policy

## Reporting a vulnerability

Please disclose responsibly — **do not open a public issue** for a security problem.

Report via **<https://opendpp-node.eu/security>** (or the contact form at
<https://opendpp-node.eu/contact>). Include what you found, how to reproduce it, and the impact. We'll
acknowledge your report and keep you updated; please allow reasonable time to remediate before any
public disclosure.

## Scope

These **public** repositories contain **no secrets** and no production keys — only open schemas,
public sample credentials (signatures over synthetic demo data), the OpenAPI contract, and a
validator. The highest-priority reports concern the **running service** (signing, key custody, the
API) — report those via the channel above. A secret accidentally committed to a public repo would
itself be a finding; please report it the same way.
